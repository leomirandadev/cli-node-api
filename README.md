# cli-node-api CLI

## How to use 
**Install cli with:**
```shell
$ yarn link
```
**For create a new api project**
```shell
$ cli-node-api init nameProject
```

If you want to set a database comunication, add --db after the project name, like:

```shell
$ cli-node-api init nameProject --db
```
